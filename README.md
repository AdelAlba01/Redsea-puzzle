# Red Sea Explorer Slide Puzzle

A mobile-friendly HTML slide puzzle game for the Red Sea Destination.

## Files
- `index.html` — main game page
- `style.css` — design and responsive layout
- `script.js` — puzzle logic, timer, moves, win screen
- `assets/redsea-resort.jpg` — game image

## How to run
Open `index.html` in any browser.

## How to publish permanently
### Netlify
1. Go to Netlify.
2. Drag and drop this entire folder.
3. Copy the public link.

### GitHub Pages
1. Create a GitHub repository.
2. Upload these files.
3. Enable GitHub Pages from repository settings.
